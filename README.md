## Features
I made basic app fundamentals.
Current features:

- add lesson
- add notes

- delete lesson
- delete note
- delete exercise
- delete subject
- editor
- viewing notes
- user management
- editing content
 
## Starting server
To start development server:
```shell
npm run dev
```
To start a server:
```shell
npm start
```
